# Sistema OANSE - Backend

Backend em Java com Quarkus + Hibernate Panache, criado para gerenciar:

- Cadastro de oansistas
- Designação para líderes
- Registro de presença
- Histórico por ano
- Futuro módulo de seções/manuais

Tech stack:

- Java 21
- Quarkus
- Hibernate Panache (Active Record)
- PostgreSQL (dev via Docker)
